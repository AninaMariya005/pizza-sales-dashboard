                                                🍕 Pizza Sales Dashboard

🧾**1. Pizza Sales Report**

🍕 Pizza Sales Insights Dashboard: Order Trends & Revenue Analytics
An interactive Power BI dashboard designed to analyze pizza sales performance, customer behavior, and revenue trends.

🎯**2. Purpose**

This dashboard helps businesses analyze sales performance, track key metrics, and uncover customer ordering patterns. It supports data-driven decisions in marketing, operations, and product strategy.

🛠️**3. Tech Stack**

📊 Power BI Desktop – Data visualization & dashboard creation

📂 Power Query – Data cleaning and transformation

🧠 DAX – Calculated measures (KPIs, revenue, averages)

🔗 Data Modeling – Relationships between tables

📁 File Formats – .pbix, .png

📂 **4. Data Source**

Source: Sample pizza sales dataset
Includes:
Order details (date, time, quantity)
Pizza categories (Classic, Supreme, Veggie, Chicken)
Pizza sizes (S, M, L, XL)
Pricing and revenue

⭐ **5. Features / Highlights**

🔍 Business Problem
Businesses often struggle to identify:
Top-performing products
Peak sales periods
Customer ordering patterns

🎯 Goal of the Dashboard

Track KPIs
Analyze trends
Identify best-selling products
Support decision-making

📊 Walkthrough of Key Visuals 

🔢 KPIs: Revenue (669K), Orders (17K), Avg Order Value (38.33), Pizzas Sold (41K)

📅 Daily Trend: Peak orders on weekends (Fri–Sat)

📈 Monthly Trend: Highlights seasonal demand patterns

🍕 Category Analysis: Classic category leads in sales

📏 Size Analysis: Large pizzas contribute the most

🏆 Top Products: Identifies best-selling categories

💡 Business Impact & Insights-
📈 Optimize staffing during peak days
🎯 Focus marketing on high-performing products
📦 Improve inventory planning
📊 Enable better forecasting
